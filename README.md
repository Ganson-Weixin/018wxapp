## 5.1学生前端功能模块

![](/md/blog.014.png)

图5-1学生登录界面图

![](/md/blog.015.png)

图5-2首页功能界面图

实习公告，在实习公告页面可以填写公告标题、发布时间、发布人、封面等信息进行提交，如图5-3所示。

![](/md/blog.016.png)

图5-3实习公告界面图

![](/md/blog.017.png)

图5-4实习信息界面图

![](/md/blog.018.png)

图5-5学生打卡界面图
